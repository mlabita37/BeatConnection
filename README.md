###Beat Connection
A Node.js, Express, and Mongoose drum sequencer. Create original beats right in your browser!

![](https://media.giphy.com/media/Ej3jPAdYI1xx6/giphy.gif)

###System dependencies
bcryptjs: 2.3.0,
body-parser: 1.15.0,
cookie-parser: 1.4.1,
ejs: 2.4.1,
express: 4.13.4,
mongoose: 4.4.4
